Fundamentos de visualización de datos en SAS

PROC SGPLOT como herramienta principal:

  • Procedimiento moderno para visualizaciones
  • Genera automáticamente ejes y leyendas
  • Opciones para personalizar títulos y estilos
  • Ideal para exploración rápida
  • Perfecto para reportes ejecutivos

Tipos de gráficos y aplicaciones:

  • VBAR/HBAR: Distribuciones de categorías
  • SERIES: Tendencias temporales
  • VBOX/HBOX: Dispersión de variables numéricas
  • HISTOGRAM: Distribuciones de frecuencia
  • SCATTER: Relaciones entre variables

Principio: Seleccionar el tipo correcto es crucial para comunicar el mensaje

Mostrar cómo ODS GRAPHICS ON habilita automáticamente los gráficos. Enfatizar diferencias con visualización básica.

Creación de agregados para visualización

Agregados temporales:

/* Serie temporal: ventas mensuales */
proc sql; 
    create table work.ventas_mensual as
    select mes_venta as mes
         , sum(monto_total) as monto_mes
    from work.ventas_prep
    group by mes_venta
    order by mes_venta; 
quit;

Agregados por dimensiones de negocio:

/* KPI por categoría */
proc sql; 
    create table work.kpi_categoria as
    select categoria
         , sum(monto_total) as monto_total
         , count(*) as n_ventas
    from work.ventas_cli_prod_vend 
    group by categoria 
    order by monto_total desc; 
quit;

Principio: Crear datasets específicos para cada visualización mejora rendimiento

Mostrar cómo usar datos integrados de bloques anteriores. Enfatizar importancia de agregación previa.
PROC SGPLOT
BLOQUE 5

Gráficos de series temporales con PROC SGPLOT

Configuración inicial:

/* Habilitar gráficos SG */
options nocenter validvarname=v7;
ods graphics on;

Serie temporal básica:

/* Ventas mensuales */
proc sgplot data=work.ventas_mensual;
    title "Ventas mensuales (total)";
    series x=mes y=monto_mes / markers;
    yaxis label="Monto" grid; 
    xaxis label="Mes";
run; 
title;

Series múltiples:

/* Por país y mes */
proc sgplot data=work.ventas_mes_pais;
    title "Ventas mensuales por país";
    series x=mes y=monto_mes / group=pais;
    yaxis label="Monto" grid;
    xaxis label="Mes";
run; 
title;

Gráficos de barras: comparación de categorías

Barras verticales ordenadas:

/* Por categoría */
proc sgplot data=work.kpi_categoria;
    title "Ventas por categoría";
    vbar categoria / response=monto_total 
                     datalabel 
                     categoryorder=respdesc;
    yaxis label="Monto" grid;
run; 
title;

Barras horizontales (Top N):

/* Top 10 vendedores */
proc sgplot data=work.kpi_vendedor(obs=10);
    title "Top 10 vendedores por monto";
    hbar vendedor / response=monto_total 
                    datalabel 
                    categoryorder=respdesc;
    xaxis label="Monto" grid;
run; 
title;

Opciones clave: DATALABEL muestra valores, CATEGORYORDER=RESPDESC ordena por monto

Enfatizar diferencia entre VBAR y HBAR. Mostrar opciones de ordenamiento y etiquetas de datos.
Diferencia entre formato permanente (DATA step) vs temporal (PROC step). WHERE vs IF.

Gráficos de distribución y dispersión

Boxplots (cajas):

/* Distribución por categoría */
proc sgplot data=work.ventas_cli_prod_vend;
    title "Distribución de monto por categoría";
    vbox monto_total / category=categoria;
    yaxis label="Monto por transacción" grid;
run; 
title;

Histogramas:

proc sgplot data=work.ventas_cli_prod_vend;
    title "Histograma de monto_total";
    histogram monto_total; 
    density monto_total;
    xaxis label="Monto"; 
run;

Diagramas de dispersión:

/* Cantidad vs monto */
proc sgplot data=work.ventas_cli_prod_vend;
    title "Cantidad vs Monto (scatter)";
    scatter x=cantidad y=monto_total / 
            transparency=0.1;
    xaxis grid; 
    yaxis grid;
run; 
title;

Aplicación: Analizar dispersión, outliers y relaciones entre variables

Boxplots muestran cuartiles y outliers. Histograma + densidad da distribución completa. Scatter para correlaciones.

Principio: Detectar inconsistencias antes de que afecten el análisis

Gráficos con paneles y facetas múltiples

/* Ventas por categoría por vendedor */
proc sgpanel data=work.ventas_cli_prod_vend;

KPIs
BLOQUE 5

Cálculo de KPIs y métricas ejecutivas

Agregación por categoría:

proc sql;
    create table work.kpi_categoria as
    select categoria,
           sum(monto_total) as monto_total,
           count(*) as n_ventas,
           sum(cantidad) as unidades,
           calculated monto_total / calculated unidades 
               as precio_prom format=dollar8.2
    from work.ventas_cli_prod_vend
    group by categoria
    order by monto_total desc;
quit;

KPI por vendedor:

proc sql;
    create table work.kpi_vendedor as
    select vendedor,
           sum(monto_total) as monto_total format=dollar12.2,
           count(*) as n_ventas,
           sum(cantidad) as unidades,
           calculated monto_total / calculated n_ventas 
               as ticket_promedio format=dollar8.2
    from work.ventas_cli_prod_vend
    group by vendedor
    order by monto_total desc;
quit;

Aplicación: Métricas listas para dashboards ejecutivos

KPIs se calculan con SQL para mayor flexibilidad y se formatean para presentación.
ODS Export
BLOQUE 5

Exportación de gráficos y reportes

Excel multi-hoja:

ods excel file="paquete_b5.xlsx" 
    style=HTMLBlue options(frozen_headers='on');

ods excel options(sheet_name='Resumen');
proc report data=work.ventas_mensual nowd; 
    columns mes monto_mes; 
run;

ods excel options(sheet_name='Categorias');
proc report data=work.kpi_categoria nowd; 
    columns categoria monto_total n_ventas; 
run;

ods excel close;

PDF ejecutivo:

ods pdf file="reporte_b5.pdf" 
    style=Pearl notoc;
    
title "Reporte ejecutivo — Bloque 5";

proc sgplot data=work.ventas_mensual; 
    series x=mes y=monto_mes / markers; 
run;

proc report data=work.kpi_vendedor(obs=10) nowd; 
    columns vendedor monto_total n_ventas; 
run;

ods pdf close; title;

Aplicación: Reportes listos para compartir con stakeholders

ODS permite crear reportes profesionales en múltiples formatos de una sola ejecución.
